# COVID-19 Dashboard (Power BI)

This project is a data visualization dashboard created using Microsoft Power BI to analyze COVID-19 data across different states.

## Project Overview
The dashboard provides insights into:
- Total confirmed cases
- Total recoveries
- Total deaths
- Daily average cases
- State-wise comparison

## Key Insights
- Total cases are high across all states
- Recovery count is higher than deaths
- Death rate is low
- Data is evenly distributed across states

## Dashboard Features
- KPI cards
- Bar charts
- Donut chart (year-wise)
- Comparison charts
- Filters for state selection

## Tools Used
- Microsoft Power BI

## How to Use
1. Download the .pbix file
2. Open in Power BI Desktop
3. Explore dashboard
